To use JSF we need to set up development environment for JSF.

We need to make sure that the Java Development Kit (JDK) installed along with the JRE.

To get help with the JSF installations you can view the video in the link  Setup_video

This video will take you through all the steps in installing the JDK.

The following softwares are to be installed for JSF environment

  1. Java Application Server – Tomcat
  2. Java Integrated Development Environment (IDE) – Eclipse
  3. JSF JAR Files
  4. Connect Eclipse to Tomcat

Java Application Server

There are many Java Application Servers available, like Tomcat, GlassFish, JBoss, WebLogic, and WebSeer. However, tomcat is the most used server.

Java Integrated Development Environment (IDE)

We need to install either Eclipse or Netbeans to work within.

JSF JAR Files

Download the JSF JAR files to be able to create your own applications. JSF is an API, and it has a number of classes and interfaces that you have to use in your projects.

You can manually download the JAR files from Oracle’s website and manually include these in your projects, and the other way is to automatically download the JSF JAR files in Eclipse

In the Helloworld example section, It will be shown how Eclipse can automatically downloaded and included in your project.

Connect Eclipse to Tomcat

Lastly, you will need to connect Eclipse to Tomcat. This will allow you to start the Tomcat server from the Eclipse IDE. This configuration will make your life easier during development and testing.
