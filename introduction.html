Java Server Faces (JSF) is a Java-based web application framework intended to simplify development, integration of web-based user interfaces. Java Server Faces is a standardized display technology which was formalized in a specification through the Java Community Process.

This tutorial helps to understand the important concepts to begin with JSF web application development.
